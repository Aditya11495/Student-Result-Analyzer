# Student Result Analyzer

A data analysis project that evaluates student performance and generates insights using Python.

#Features
- Cleaned real-world dataset
- Calculated student averages
- Identified class topper
- Generated student-level insights
- Visualized subject-wise performance

# Tech Used
- Python
- Pandas
- Matplotlib

# Project Goal
To practice real-world data analysis workflow:
Data Cleaning → Analysis → Visualization → Insights.
